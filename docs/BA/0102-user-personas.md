# User Personas - SiswaPresensi

## Persona 1: Budi Santoso (Guru)

**Role:** Guru Matematika

**Demografi:**
- Usia: 35 tahun
- Pekerjaan: Guru Matematika di SMA Negeri 1 Jakarta
- Lokasi: Jakarta
- Status: Menikah, 2 anak

**Goals:**
- Input presensi siswa dengan cepat dan akurat
- Tidak ingin membuang waktu memanggil nama satu per satu
- Ingin tahu siswa yang tidak sering hadir
- Ingin approve/reject request izin/sakit dengan mudah
- Ingin melihat laporan presensi secara real-time

**Pain Points:**
- Menghabiskan 10-15 menit per kelas untuk presensi manual
- Sering salah catat atau terlewat saat memanggil nama
- Tidak tahu real-time status kehadiran siswa
- Request izin/sakit datang via WA/telepon, sulit track
- Laporan presensi hanya ada saat rapor bulanan

**Behaviors:**
- Mengajar 6 kelas/hari (XI IPA 1, XI IPA 2, XI IPA 3, XII IPA 1, XII IPA 2, XII IPA 3)
- Aktif menggunakan smartphone dan laptop
- Terbiasa dengan aplikasi web
- Menginginkan efisiensi dalam pekerjaan sehari-hari

**Tech Savviness:** Sedang
- Nyaman menggunakan aplikasi web
- Terbiasa dengan browser
- Mengerti dasar-dasar teknologi

## Persona 2: Ahmad Rizky (Siswa)

**Role:** Siswa Kelas XI IPA 1

**Demografi:**
- Usia: 16 tahun
- Pekerjaan: Pelajar SMA Negeri 1 Jakarta
- Lokasi: Jakarta
- Status: Single

**Goals:**
- Ingin presensi mudah dan cepat
- Tidak ingin antre untuk presensi
- Ingin tahu jadwal pelajaran hari ini
- Ingin request izin/sakit dengan mudah
- Ingin melihat riwayat presensi

**Pain Points:**
- Harus menunggu dipanggil nama satu per satu
- Takut kelewatan saat guru memanggil nama
- Tidak tahu jadwal pelajaran hari ini
- Request izin/sakit harus lewat orang tua atau WA guru
- Tidak bisa lihat riwayat presensi sendiri

**Behaviors:**
- Aktif di media sosial
- Memiliki smartphone dengan koneksi internet
- Lebih suka hal yang praktis dan cepat
- Sering request izin untuk acara keluarga

**Tech Savviness:** Tinggi
- Nyaman menggunakan smartphone dan web apps
- Terbiasa dengan QR code
- Memiliki smartphone dengan koneksi internet stabil

## Persona 3: Siti Aminah (Orang Tua)

**Role:** Orang Tua Ahmad Rizky

**Demografi:**
- Usia: 42 tahun
- Pekerjaan: Ibu rumah tangga
- Lokasi: Jakarta
- Status: Menikah, 2 anak

**Goals:**
- Ingin tahu apakah anak hadir di sekolah
- Ingin request izin/sakit untuk anak dengan mudah
- Ingin monitor kehadiran anak secara real-time
- Ingin terima notifikasi saat anak tidak hadir
- Ingin lihat riwayat kehadiran anak

**Pain Points:**
- Tidak tahu real-time apakah anak hadir di sekolah
- Request izin/sakit harus WA guru atau datang ke sekolah
- Tidak bisa monitor kehadiran anak
- Tidak terima notifikasi saat anak tidak hadir
- Hanya tahu kehadiran anak saat rapor bulanan

**Behaviors:**
- Mengurus rumah tangga dan anak-anak
- Aktif di WhatsApp
- Menginginkan anak rajin sekolah
- Khawatir jika anak sering tidak hadir

**Tech Savviness:** Sedang
- Nyaman menggunakan smartphone
- Terbiasa dengan WhatsApp
- Mengerti dasar-dasar aplikasi web

## Persona 4: Admin Sekolah (Super Admin)

**Role:** Super Admin Sistem

**Demografi:**
- Usia: 40 tahun
- Pekerjaan: IT Administrator SMA Negeri 1 Jakarta
- Lokasi: Jakarta
- Status: Menikah, 1 anak

**Goals:**
- Mengelola seluruh sistem sekolah
- Mengelola users (guru, siswa, orang tua)
- Memantau performa sistem
- Melihat laporan dan analitik
- Mengatur pengaturan sistem

**Pain Points:**
- Terlalu banyak sistem yang harus dikelola manual
- Sulit tracking perubahan data dan aktivitas user
- Tidak memiliki visibility penuh terhadap aktivitas sistem
- Tidak ada audit trail untuk aktivitas kritis
- Laporan manual memakan waktu

**Behaviors:**
- Memeriksa sistem setiap hari
- Membuat laporan rutin untuk kepala sekolah
- Berhati-hati dalam memberikan akses dan permission
- Mengikuti best practice security dan compliance

**Tech Savviness:** Tinggi
- Ahli dalam administrasi sistem dan database
- Nyaman dengan dashboard dan tools teknis
- Memahami konsep security dan data protection
